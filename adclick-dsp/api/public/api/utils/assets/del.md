　
# 删除素材
---
### 简要描述：
> 删除素材库的一个素材，正在被广告创意使用的素材不能删除

　　　　

### 请求地址：```/v3/utils/assets/del```

### 请求方式：POST

### 格式 ：x-www-form-urlencoded
　

### 参数列表

 参数名 | 必选 | 数据类型 | 说明 
 :------ | :----:| :--------: |:---- 
 asset_id|是|String|素材id


　

### 返回示例
```
{
    "code": 0,
    "message": "",
    "data": {}
}
```
　

### 返回参数说明

参数名 | 类型 | 说明
:---   |:---: |:---
code | int | 请求码，请求成功则code=0
message | String | 错误消息，请求成功则消息为空
　

### 备注
>更多返回请求码请看错误代码描述
