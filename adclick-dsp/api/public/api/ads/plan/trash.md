
#tarsh广告计划
---
### 简要描述：
> trash广告计划，设置状态清楚位
> dsp_adlib:plans

### 请求地址：```/v3/ads/plan/trash```

### 请求方式：POST

### 格式： x-www-form-urlencoded

### 参数列表

|参数名 | 必选 | 数据类型 | 说明|
|:---   | :--: | :------: | :---|
|user_id|是|BidInt|广告主id
|plan_id|是|BigInt|计划id


### 返回示例
```
{
    "code": 0,
    "message": "",
    "status":"int" 成功：0，失败：-1
}
```
